# MediTrack

## Project Overview
MediTrack is a comprehensive medical tracking application designed to help users manage their health records efficiently. The app allows for the easy input and retrieval of medical data, medication schedules, and health metrics.

## Features
- User authentication and profile management
- Secure storage of medical records
- Medication reminders
- Visual health metrics tracking
- Data export options

## Tech Stack
- **Frontend:** React, Redux
- **Backend:** Node.js, Express
- **Database:** Postgresql

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Yashasri-g/MediTrack.git
   ```
2. Navigate to the project directory:
   ```bash
   cd MediTrack
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the application:
   ```bash
   npm start
   ```

## Usage
- After starting the application, visit `http://localhost:3000` in your browser.
- Create an account or log in if you already have one.
- Begin entering your health records and setting up medication reminders.
